# ML Revision — Machine Learning S4

Plateforme interactive de révision pour le module Machine Learning S4.  
Ce projet est une application web 100% front-end (HTML, CSS, JavaScript) permettant de réviser efficacement à travers différents formats pédagogiques.

---

## Description du projet

Cette plateforme propose une expérience moderne et intuitive pour préparer les examens de Machine Learning. Elle regroupe cours, exercices, QCM et simulations d’examen dans une seule interface.

---

## Fonctionnalités principales

- Navigation fluide avec une barre latérale
- Mode sombre et mode clair
- Timer intégré pour s’entraîner
- Cours théoriques organisés par modules
- Exercices guidés avec explications
- QCM interactifs avec correction immédiate
- Flashcards pour la mémorisation
- Simulations d’examen (complet et rapide)
- Interface responsive (compatible mobile et desktop)

## Installation et utilisation

### 1. Cloner le projet

```bash
git clone https://github.com/Samah-boudallaa/ML_REVISION.git
```
### 2. Ouvrir l’application

Ouvrir le fichier suivant dans un navigateur :
```bash
machine_learning_revision.html
```
Aucune installation supplémentaire n’est nécessaire.
